GW2CL Sign


Quatre paramètres requis pour générer une signature:
  • joueur : Le nom du joueur connecté. (ex: Liath)
  • arme : Le nom de l'arme crée. (ex: Aurore)
  • avancement : L'avancement en cours, sans le signe pourcentage %. (ex: 55.1)
  • fond : Indique quel fond sera utilisé. (ex: 2)
